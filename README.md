# Facets UI extend React Native code

###### YARN

```
yarn install
```

###### NPM

```
npm install
```

###### IOS

```
cd ios && pod install && cd ..
```

```
yarn ios
```
###### or
```
npm run ios
```

###### ANDROID
* open the emulator in Android Studio before run the terminal command
* run in the terminal
```
yarn android
```
###### or
```
npm run android
```
